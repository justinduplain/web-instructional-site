6) Update the CSS

Before adding the lightbox images to the page, we'll want to implement the CSS that hides the lightbox and larger images until their :target pseudo class is activated. We'll also add a few other style updates. 

  • Open your yoga.css file in a text editor

6.a.) Add lightbox styles

We'll want our lightbox to remain hidden until the user clicks on an image, so we're going to set styles to display:none when the page first loads. We also want to make sure the lightbox covers the entire page and is positioned above all the other content in the page.

Add the below styles to your CSS file in the desktop section (BEFORE the media queries)


.lightbox {

    /** lightbox does not display initially **/
    display: none;

    /** Position and style of lightbox. It covers all of the page starting at the top left corner **/
    position: fixed;
    width: 100%;
    height: 100%;
    text-align: center;
    top: 0;
    left: 0;

    /** Places the lightbox above all other content **/
    z-index: 999;

    /** Adds a black background at 80% opacity **/
    background: rgba(0,0,0,0.8);
}

Save your file. 

6.b.) Style the lightbox content

We want our images and the video to display nicely in the lightbox, even if they are too large for the screen so we'll add some styles to make sure it fits to the viewport and has some space at the top.

.lightbox img {
    max-width: 75%;
    margin-top: 2%;
}

.lightbox iframe {
    /** Adds a top margin for the YouTube iFrame **/
    margin-top: 15%;
}

Save your file. 

6.c.) Style the pseudo classes

Here we use the :target pseudo class to show the lightbox and image when its ID is selected--when the appropriate thumbnail is clicked or tapped. We also remove the default browser image outline. 


.lightbox:target {
    display: block;
    transition: .5s ease;
    outline: none;
}

We'll also use use the :hover pseudo class to give some interaction to the thumbnails, this will reduce the image opacity when hovering over them.  

.thumb:hover {
    opacity: .8;
    transition: .1s ease;
}

Save your file.
